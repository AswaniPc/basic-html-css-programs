A Cellular Network Planning Technique to Minimize Exposure to RF Radiation

Introduction

Cellular communications use the cellular network as its infrastructure. Service coverage area is divided into smaller areas called cells. Each cell is served by a base station (BS). A base station is connected to the mobile switching center (MSC). MSC connected to the public switched telephone network (PSTN).

Problem Definition

Given a set S of sites and a planned cellular network T with a fixed cell size and network orientation. To find the position C to deploy the planned network such that the minimal network distance between the site set S and the deployed network T(C) is maximized.

Solution

Based on a new geometric diagram called the nearest-point umbrella diagram. S ={P0 , P1 , ..., Pn } set of n planar points. T is thenetwork pattern. The nearest-point umbrella diagram of S under T divide the plane into a set of n regions, UT (Pi , S) is the locus of points that have smaller network distance to T (Pi ) than to T (Pj ) for all j = i. UT (Pi , S) is the umbrella region of Pi . If planned network deployed at any point within umbrella region of Pi then Pi will have a smaller distance to deployed network than Pj . To maximize the distance between Pi and deployed network, Planned network deployed on the point at the contour of umbrella region (Ci ). that Point has largest distance to Pi . di is the Network distance between Ci and Pi . Compute di for all Pi . Ck is the point of deployment of T.

Conclusion

Nearest point umbrella diagram of n sites computed in O(n 2logn) time. Based on the umbrella diagram Minimal exposure problem of size n can be solved in O(n 2 logn) time.

Limitations

High complexity of algorithms to compute the umbrella diagrams.
